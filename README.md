# Gen-Z Career Aspirations Analysis

## Overview
Analyzed 3,611 Gen-Z survey responses to uncover trends in work environment 
preferences, salary expectations, career aspirations, and values alignment.

## Tools Used
SQL | Microsoft Excel | Power BI | Power Query

## Project Structure
- /data — cleaned dataset
- /sql — SQL queries answering 5 business questions
- /dashboards — Excel and Power BI dashboard files
- /presentation — final project presentation

## Key Findings
- Hybrid work dominates as the preferred environment
- Most respondents expect salaries of 31K–40K early career
- Strong values alignment preference — X% want mission-driven employers
